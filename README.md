💳 Payooo Mobile Banking App
A fully functional fintech-style web app built using HTML, Tailwind CSS, and JavaScript.
This project simulates a real mobile banking experience with interactive features like transactions, transfers, and bill payments.



✨ Features
💰 Add Money
💸 Cash Out
🔄 Transfer Money
🎁 Bonus System (Coupon Based)
🧾 Pay Bills
📊 Transaction Dashboard
🔐 PIN Verification System (Demo: 1234)
💾 LocalStorage-based Balance Management
🎯 Dynamic Active Navigation UI
🛠️ Technologies Used
HTML5
Tailwind CSS
JavaScript (DOM Manipulation, Events, LocalStorage)

⚡ How It Works
Balance is stored using localStorage
Each feature updates balance dynamically (Add, Withdraw, Transfer)
PIN verification ensures secure actions
Forms include validation for correct input
Navigation system highlights active feature dynamically
